# Sugary World

Sugary World é um jogo de plataforma em C, inspirado em Mário, onde o jogador controla um personagem que precisa saltar entre plataformas, coletar itens e evitar obstáculos para avançar nos níveis.

## Funcionalidades

- Movimentação lateral com setas do teclado
- Pulo com física realista
- Diferentes níveis de dificuldade

## Controles

- Setas: movimentação do personagem
- ESC : sair do jogo

## Requisitos

- Compilador C (GCC recomendado)
- Biblioteca Raylib
- Sistema operacional: macOS

## Como Compilar e Rodar

```a
# Clonar repositório
git clone https://github.com/nibrito20/sugary_world.git
cd sugary_world

# Compilar o jogo
make

# Rodar o jogo
make run

# Limpar arquivos compilados
make clean
```

⚠️ *Certifique-se de ter o GCC e a Raylib instalados antes de rodar make.*
